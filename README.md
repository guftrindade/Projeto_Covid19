# Projeto_Covid19

Olá, este é apenas um simples projeto de visualização para praticar os conceitos de Ciência de Dados que vou aprendendo ao longo da minha jornada de aprendizado.

Uma coisa que gosto sempre de mencionar é no poder que a Estatística possui na tomada de decisão, porém ela tem a capacidade de tramistir uma infinidade de informações, 
caminhos diferentes e mesmo assim resultados verdadeiros. Um exemplo recente e atual deste fato é o número de casos de Covid-19 no Brasil. 

Em um cenério pessimista (usado principalmente pelos veículos de imprensa), o Brasil possui até o dia de hoje (18/03/2021) o total de 11.780.820 casos e 287.499 mortes de Covid-19. 
Comparando este número com outros países, o Brasil é o segundo país com maior número de casos e mortes no mundo. 

Analisando de uma maneira diferente, desta vez em um cenário otimista (usado principalmente pelo governo), podemos levar em consideração que o Brasil hoje possui 
aproximadamente 210 milhões de habitantes (IBGE, 2021), logo, uma população maior tende a acarretar um maior número de casos e consequentemente um maior número de mortes.
Mas enfim, ao encontrarmos o índice de contamição (Número de casos/Número de habitantes) e o índice de mortalidade (Número de Mortes/Número de Habitantes), podemos entender que 
o Brasil está menos pior que muitos outros países.

Portanto, para analisar de forma visual, realizei este comparativo utilizando Power BI para criação do Dashboard.
